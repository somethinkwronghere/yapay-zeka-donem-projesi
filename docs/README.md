# Docs

Projeye ait dokümanlar.

| Dosya | Açıklama |
|-------|----------|
| `proje_kilavuzu.pdf` | Dersin resmi dönem projesi kılavuzu (hoca tarafından paylaşıldı). |
| `project_plan.md` | Haftalık çıktı planı, modelleme stratejisi ve risk analizi. |
| `prior_work.md` | Proje öncesi yapılan 13 Colab defterinin (Chronos, Diffusion, TimeGAN aileleri) envanteri ve karşılaştırması. Rapor "Önceki Çalışmalar" bölümü için referans. |
| `images/` | Rapor ve sunum için grafikler. |
| `report/` | Final teknik raporunun (IEEE formatında PDF) saklandığı yer. |
